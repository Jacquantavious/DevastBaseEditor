⚙ Base Editor Controls

Navigation

  • Scroll Wheel — Zoom in/out at cursor
  • Middle Click + Drag — Pan the map
  • Right Click + Drag — Pan the map
  • WASD / Arrow Keys — Pan the map
  • R — Rotate selected block (directional blocks only)
  • H — Toggle this help window

Placement Modes

  • Click — Left-click to place; click occupied cell to remove
  • Line — Click two points to draw a line of blocks
  • Clone — Click two points to select area; then click to paste
  • Delete — Click two points to erase an area

Map Code

  • Copy Code — Generates and copies code to clipboard
  • Load — Paste code into the box, then click Load
  • Clear — Remove all placed blocks
